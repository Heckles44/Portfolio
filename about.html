About Me

My name is David and I’m a self-taught web developer with a background in civil engineering with over 6 years professional experience.

I’ve been learning full-stack web technologies outside of my full-time employment, using various online resources. Prior to completing my bachelors degree in civil engineering I was enrolled in a bachelors of electronic engineering at La Trobe University, where I completed units in Java and C programming languages.

Since the outbreak of COVID-19, I’ve been able to focus more intensely on programming. I’m now well-versed in the main front end technologies including HTML, CSS, Sass, JavaScript, React.js. More recently, I have been applying my focus to back end technologies such as PHP, Node.js, Express, PostgreSQL and MongoDB. Additionally to the above, I have strong familiarity with utilising the command line (bash), version control with Git and GitHub, and API testing with Postman.
